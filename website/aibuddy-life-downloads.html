Installation Help

🍎 Mac Installation

If you see "AIBuddy is damaged and can't be opened":

Run this command in Terminal:

xattr -cr /Applications/AIBuddy.app

Alternative: Right-click AIBuddy.app → "Open" → click "Open" in the dialog.

Or: System Settings → Privacy & Security → "Open Anyway"

🪟 Windows Installation

If SmartScreen shows "Windows protected your PC":

  1. Click "More info"
  2. Click "Run anyway"

🐧 Linux Installation

chmod +x AIBuddy-*.AppImage && ./AIBuddy-*.AppImage
